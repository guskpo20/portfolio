## Concept

A tribute to the 80s classics. The goal was to create a lightweight game engine without external dependencies.

### Features

*   **Particle System:** Explosions and visual effects rendered at 60fps.
*   **High Scores:** Local persistence of scores.
*   **Audio API:** Real-time synthesized sound effects.

### Demo

You can play the demo directly in the browser. The code is 100% Vanilla JS.
